# Echoes-we-forgot
An interactive digital museum preserving disappearing everyday sounds and lost artworks — built with React, Wonder (AI design), and Cursor/Codex via MCP for the Wonder x Contra Design Challenge.


# 🎧 Echoes We Forgot

**A digital museum for the sounds and sights the world is quietly leaving behind.**

Every generation loses a few sounds it never meant to lose — a dial-up modem, a school bell, the whir of a VHS rewinding. *Echoes We Forgot* is an interactive museum that archives them before they're gone completely, pairing each fragment with the story of why it faded.

**https://echoes-we-forgot.netlify.app/** · **[🎨 Wonder Design File](#)** · **[🎥 Walkthrough Video](#)**

---

## What it is

Two parallel museums, one shared idea:

- **🔊 The Sound Museum** — Listen to 8 disappearing everyday sounds (a railway announcement, a Nokia ringtone, a modem dialing) and guess what they are. Every correct answer recovers a fragment of memory and a short story of why that sound went quiet.
- **🎨 The Art Museum** — The same mechanic, applied to art history: identify 8 masterpieces and the artists behind them.

Both end in a scored results card — star rating, a shareable summary, and a downloadable keepsake.

## Why it exists

Built for the **Wonder × Contra Design Challenge**, with a simple goal: don't just design something pretty — design something that means something, using Wonder end-to-end and proving the design-to-code pipeline actually works.

## How it was built

| Stage | Tool |
|---|---|
| Visual design, layout, copy, retro UI system | **Wonder** (Chat + shader/blob backgrounds) |
| Design → real code handoff | **Wonder MCP → Cursor & Codex** |
| Interactive logic (quiz state, audio, scoring) | Cursor / Codex, built on the exact Wonder-sourced styling |
| Frontend | React + Tailwind CSS + Vite |

The landing page mirrors a clean, modern editorial layout with a fixed, mouse-reactive ambient background. Stepping into either museum shifts the entire visual language into a retro, Nokia-inspired monochrome-meets-color interface — same soft gradient world underneath, completely different skin on top.

## Features

- 🎙️ 8 real, playable archived sounds with original short-story copy for each
- 🖼️ 8 real public-domain artworks with matching trivia
- 🔋 A "Memory Charge" progress system across both museums
- ⭐ Dynamic star-rating results screen with share + download
- 🎨 4 switchable color themes (Light, Yellow, Sky Blue, Dark)
- 🖱️ Mouse-reactive liquid blob background on the landing page
- 🧭 Full free navigation — skip, revisit, or answer in any order

## Tech Stack

`React` · `Tailwind CSS` · `Vite` · `Wonder` · `MCP` · `Cursor` · `Codex`
